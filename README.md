# Logistics Tracking Site (DEMO)

This repository contains a **demo version** of the Logistics Tracking System — a web-based application designed to demonstrate how parcel tracking and transportation management can be represented visually in a logistics workflow. (full version: <a href="https://github.com/PortoStack/logistics-tracking-system">https://github.com/PortoStack/logistics-tracking-system</a>)

It serves as a **frontend-only prototype** for educational and presentation purposes.

---

## Project Overview

The **Logistics Tracking Site** was created to showcase the basic concept of a logistics management system.  
It simulates parcel tracking, route assignment, and delivery progress using static JSON data.

This demo focuses on **visualizing system flow and UI interaction**, not on backend or real-time processing.

---

## Main Features

- **Parcel Tracking** — Display parcel information and delivery status.  
- **Route Simulation** — Show routes between origin and destination points.  
- **Driver & Vehicle Overview** — Present transport resources.  
- **Demo Data Visualization** — Use sample JSON files for demonstration.
